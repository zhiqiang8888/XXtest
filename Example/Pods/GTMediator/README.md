# GTMediator

GTMediator是根据[BeeHive](https://github.com/alibaba/BeeHive)修改的，规范了自己的命名规范，增加了自己的规则，跳转跳转方式。


## 安装

你可以在 Podfile 中加入下面一行代码来使用 GTMediator

```ruby
pod 'GTMediator'
```
